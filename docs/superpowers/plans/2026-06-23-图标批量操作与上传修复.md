# 图标批量操作与上传修复 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 修复批量上传中文文件名乱码、缩略图显示不完整，并新增图标批量删除与迁移所属文件夹。

**Architecture:** 后端新增批量更新 API，复用现有 SQLite 与 Express 结构；前端在卡片列表上增加多选状态和批量操作栏；样式仅补充缩略图完整显示与批量栏样式，不重构页面布局。

**Tech Stack:** Node.js、Express、multer、better-sqlite3、原生 HTML/CSS/JavaScript、node:test。
